Class name IriTemplateMapping
Obligation Optional
URI

hydra:IriTemplateMapping
Subclass Of hydra:Resource
Usage note This maps a variable used in the template to a property and may optionally specify whether that variable is required or not. The syntax of the template literal is specified by its datatype and defaults to the [RFC6570] URI Template syntax, which can be explicitly indicated by hydra:Rfc6570Template.
